# avelina, per sea

My personal website — a portrait homepage, an engineering portfolio,
a literary section (*Weaver's Cradle*), a photo gallery, and an about page.

🌊 **Live:** https://avelaxonov.github.io/avelina-per-sea/

---

Plain HTML, CSS, and JavaScript — no build step, no framework.
To run it locally, open `index.html`, or serve the folder:

```bash
python3 -m http.server 8000   # then visit http://localhost:8000
```

A static server is the better way to preview, because the project pages and
poems load through `?p=` links that behave best over `http://` rather than
straight off the filesystem.
